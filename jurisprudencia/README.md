# Jurisprudencia relevante para arrendamientos

Resúmenes de sentencias del Tribunal Supremo y Audiencias Provinciales sobre materias recurrentes en arrendamientos de vivienda.

## Estado

Todos los archivos en esta sección están **pendientes** de incorporar las sentencias concretas. Las fuentes principales son:

- CENDOJ (Centro de Documentación Judicial): https://www.poderjudicial.es/search/
- Tribunal Constitucional: https://hj.tribunalconstitucional.es/
